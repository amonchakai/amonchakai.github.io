Simple Workout Policy

General information

Simple workout is a gym log for BlackBerryÂ© 10.
It was designed to records your workouts at the gym in an efficient manner enabling to spend more time on your exercise than on using the application.
You will find a friendly to use user interface, highly customizable enabling to record all kinds of exercises. The application have many default exercises, in case you would not want to enter your own exercises manually. Nevertheless, you are free to do so!
You will also be able to plot charts enabling to visually inspect your progress. It also comes with export to CSV function, enabling you to analyse your progress on your computer!

Access to the source code:

The application is an open-source project under the GNU GPL licence. You can access to the code, edit it, modify it.

Privacy:

This application do not use any Internet access (except for displaying this page which is hosted on GitHub, adding and listing issues from GitHub in the "bug report center"). All the data recorded are kept on your phone in a local database and won't be shared.

Permission:

The application will request the permission to your shared files for the export and backups functions. If you choose not to allow this permission, this will only affect the export functions. All the other functions will be unaffected.
